Buka project terbaru
================
1.  Akses https://github.com/Azlan-saja
	=> Klik Repositories
	=> Cari dan klik mobile_2_4sia8
	=> Klik "CODE" dan salin HTTPS url
3. Buka menggunakan VS CODE
	=> Klik menu "VIEW" dan pilih "Command Palette"
	=> Ketikkan "Git" dan pilih "Git: Clone"
	=> Tempel hasil salin HTTPS url diatas dan ENTER
	=> Tentukan lokasi simpan (Contoh di local disk D)
	=> Tunggu proses clone selesai.
	=> Lanjut cari file "pubspek.yml", 
	     klik kanan pilih "Get Package"
3. Pilih Emulator dan Jalankan project
	=> Klik menu "VIEW" dan pilih "Command Palette"
	=> Ketikkan "Flutter: Select Device" dan pilih "Start Emulator Flutter"
	=> Tunggu emulator berjalan dan
	=> Buka file main.dart dan Klik menu "RUN" dan pilih "Start Debug"

Buat Project Baru di VS Code
========================
1. Klik menu VIEW => Command Pallate
2. Ketikkan atau pilih => Flutter: New Project
3. Pilih => Aplication
4. Tentukan lokasi simpan project (Contoh di C)
5. Ketikkan nama project mobile_2_4sia8
6. Tunggu proses New Project nya selesai.
========================
Jalankan Project di VS Code
========================
1. Klik menu VIEW => Command Pallate
2. Ketikkan atau pilih => Flutter: Select Device
3. Pilih Chrome
